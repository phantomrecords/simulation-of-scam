Simulation of Scam

This project simulates the user interface of a real-world scam experienced by Michael Widener. It documents how scammers posed as job recruiters and mentors to manipulate victims into making unauthorized payments under false pretenses.

The simulation is based on actual chat structures, scam offer patterns, and interface behaviors used in these cyber fraud attacks. No live scam code is hosted here — this is for educational and forensic purposes only.
